Lista de dependências:
GraphViewer(./src/lib): biblioteva com os ficheiros necessários para correr o API GraphViewer.
Resources (./src/resources): Ficheiros de texto com informação acerca de vértices, edges, tags, estações de metro e grids de teste.


Compilação:
A aplicação está feita para ser corrida no terminal do Windows, porque existe instruções para limpar o terminal que o CLion não executa corretamente.
Quando aparece a API de GraphViewer o utilizador não deve fechar esta janela, o programa está feito para ser ele a fechar na altura devida.
